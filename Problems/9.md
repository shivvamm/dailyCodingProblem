# Problem Title

Given a list of integers, write a function that returns the largest sum of non-adjacent numbers. Numbers can be 0 or negative.

## Example
Input : [2, 4, 6, 2, 5]
Output: 13

Input : [5, 1, 1, 5]
Output: 10
